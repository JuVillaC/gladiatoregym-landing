# Gladiatore Gym

Sitio web propuesto a **Gladiatore Gym**, un gimnasio con estética moderna, oscura y enérgica, diseñado para transmitir fuerza, disciplina y comunidad.

## Descripción

Esta landing page fue creada como una propuesta visual y comercial para presentar la identidad del gimnasio, destacar sus planes, ubicación, horarios y canales de contacto, con una experiencia responsive y estilo premium inspirado en su presencia en Instagram.

## Características

- Diseño moderno y responsivo
- Estética fitness premium con estilo neón
- Hero principal con impacto visual
- Menú hamburguesa para móvil
- Sección de planes con modal interactivo
- Galería tipo Instagram
- Sección de ubicación con mapa embebido
- Footer con información de contacto y redes sociales
- Animaciones suaves y efectos visuales llamativos

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Google Maps Embed
